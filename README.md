# ⭐ Hollow Diamond Pattern in C

## 📌 Description
This program prints a hollow diamond star pattern using nested loops and conditional statements in C language.

---

## 💡 Pattern Output
    *
   * *
  *   *
 *     *
*********
 *     *
  *   *
   * *
    *
---

## 🧠 Concepts Used
- Nested loops
- Hollow pattern logic
- Pyramid and reverse pyramid
- Space management
- Conditional statements

---

## ⚙️ Logic

- First part prints hollow pyramid
- Second part prints reverse hollow pyramid
- Spaces are used for alignment
- Stars are printed on borders only

---

## 💻 Language Used
- C Programming Language
